# 项目概览

Wasker Vue 是一个专为追求极致体验的开发者设计的个人网站主题。它不仅是一个展示作品的画廊，更是一个表达技术思考与设计美学的空间。

## 项目链接

- **项目地址 (GitHub)**: [https://github.com/couuas/wasker_vue](https://github.com/couuas/wasker_vue)
- **在线演示**: [https://couuas.pp.ua](https://couuas.pp.ua)

## 设计思想

### 1. 沉浸式体验
采用了暗色模式作为默认基调，配合精致的排版和微交互，为访客提供沉浸式的阅读和浏览体验。

### 2. 知识图谱 (Knowledge Graph)
引入了力导向图 (Force-Directed Graph) 来可视化展示文章与项目之间的关联。这不仅酷炫，更是一种非线性的知识导航方式，让访客能在探索中发现意外的惊喜。

### 3. "少即是多"
在此项目中，我们极力克制装饰性元素的使用。每一个像素、每一行代码都有其存在的意义。我们相信良好的内容本身就是最好的设计。

## 技术栈

- **Vue 3**: 采用 Composition API，逻辑复用更丝滑。
- **Vite**: 极速的开发服务器与构建工具。
- **D3.js / 3d-force-graph**: 驱动底层的 3D 知识图谱。
- **Markdown**: 所有的内容均通过 Markdown 编写，无需数据库，部署即用。
